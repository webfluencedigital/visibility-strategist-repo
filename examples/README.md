## What was NOT in this initial battle-test

The AI citation audit (querying ChatGPT / Claude / Gemini / Perplexity with the prompt set) was scheduled for the 14-day recheck rather than completed at audit time. Future battle-tests will include the citation scorecard inline.

## How to read the PDF

Open in any PDF viewer. The structure mirrors the agent output flow:

1. **Cover** — client name, target site, plan period
2. **Executive summary** — counts + SEO/AEO snapshot + headline insight
3. **AI citation scorecard** — citation rate per platform (pending baseline in this audit)
4. **Keyword landscape** — every keyword with volume, KD, intent, current/target position
5. **Technical findings** — every issue ranked by severity
6. **Sitemap recommendations** — proposed pages with primary keyword + schema
7. **Fix pack** — P1 / P2 / P3 prioritised actions with expected impact
8. **90-day content plan** — 12 posts scheduled
9. **Contact footer** — methodology, sources, next review date
