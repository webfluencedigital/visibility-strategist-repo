01 — Executive Summary

Where you stand. What we'd change.

Two domains in scope. The legacy site (webfluence.digital) is what Google currently sees and ranks — a Vite React app with content rendered client-side. The new site (webfluence-digital.vercel.app) is a single static HTML page — fully crawlable but with no surface area to rank against. The plan: expand the new site to 10 pages, fix 8 technical issues, build entity signals (Wikidata + Crunchbase), ship 12 cluster posts in 90 days, and migrate the legacy domain when the new architecture stabilises.

06 — Fix Pack

Do these in this order.

P1 in 7 days. P2 in 30. P3 within 90. Each has expected impact — quoted, not guessed. The first three P1 fixes unlock everything downstream.

  • P1
    Expand new Vercel site to 10-page architecture
    Implement the Sitemap Recommendations. Use Vercel file-based routing or Astro/Next.js static export. Preserve the cinematic homepage; build out service, industry, pricing, resource, and case study pages. Internal linking: services up to /, industries to services + /, resources cross-linked.
    EXPECTED IMPACT: unlocks 8 keyword targets and 10 ranking surfaces. Single biggest move in the plan.
  • P1
    Add Organization + LocalBusiness + FAQPage JSON-LD schema
    JSON-LD block in <head> on /, /pricing, and every service page. Validate via search.google.com/test/rich-results before deploy. FAQPage uses the 8 Question Bank entries as the seed set.
    EXPECTED IMPACT: +15-25% AEO citation rate on pricing/services queries. Eligible for AI Overview.
  • P1
    Update title tag + meta description on /
    Replace current "WebFluence Digital — ROI-Driven Web Design & SEO for Irish & UK SMEs" with "Web Design Dublin | Live in 7 Days | WebFluence Digital" (55 chars, keyword-led). Meta description leads with "Custom websites for Dublin SMEs. Live in 7 days. Conversion-first design with SEO baked in. Starter from €997."
    EXPECTED IMPACT: incremental CTR lift on existing impressions. Aligns title with primary commercial keyword.
  • P2
    Build Wikidata + Crunchbase entity profiles
    Wikidata: type Q4830453 (business), inception, founder (Dusan Walla), area-served, official website, sameAs links. Crunchbase: organisation profile with founders, category, founding location.
    EXPECTED IMPACT: stronger entity disambiguation in Claude + Perplexity. Pre-requisite for sustained AI citation.
  • P2
    Plan legacy webfluence.digital migration to SSG
    Current Vite React SPA renders client-side. Migrate to Next.js static export or Astro before sunsetting in favour of new Vercel site. Preserve URL structure during cutover. Submit updated sitemap to GSC.
    EXPECTED IMPACT: protects existing impressions during the migration. Avoids ranking loss on cutover.
  • P2
    Run baseline AI citation audit (4 platforms × 30 prompts)
    Query ChatGPT, Claude, Gemini, Perplexity with the Question Bank prompt set. Record citation rate per platform, identify which competitors win. Update Visibility Plan with baseline scorecard.
    EXPECTED IMPACT: data foundation for AEO progress measurement. Required before P1/P2/P3 fixes can be evaluated.
  • P3
    Original case study content for AI training corpus
    Publish the Adare Clinic and Shelbourne Dental case studies with original screenshots, quoted client testimonials, and ranking/traffic deltas. Goal: become the authoritative reference for "best dental web designer ireland" in Perplexity.
    EXPECTED IMPACT: long-tail AEO citations on industry-specific best-of queries.
